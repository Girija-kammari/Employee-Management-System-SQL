# Employee Management System (SQL)

## Project Overview
This project is a SQL-based Employee Management System developed using MySQL Workbench. 
It manages employee records, departments, payroll, qualifications, and leave tracking using a structured relational database design.

## Database Design
- Designed a normalized relational schema
- Implemented Primary Keys and Foreign Keys
- Applied cascade rules to maintain referential integrity
- Ensured data consistency through constraints

## Advanced SQL Implementation
- Wrote 20+ advanced SQL queries using:
  - JOINs (INNER, LEFT)
  - Aggregate Functions (SUM, AVG, COUNT)
  - Window Functions
  - Subqueries
  - GROUP BY and HAVING clauses

## Key Insights Generated
- Top 5 highest salary employees
- Qualification-wise employee count
- Department-wise workforce distribution
- Payroll summaries and salary benchmarking
- Leave frequency analysis
